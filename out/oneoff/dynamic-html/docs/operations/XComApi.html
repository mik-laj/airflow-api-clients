getXComEntry

This endpoint allows specifying `~` as the dag_id, dag_run_id, task_id to retrieve XCOM entries for for all DAGs, DAG Runs and task instances.

URL

http://localhost/api/v1/dags/{dag_id}/dagRuns/{dag_run_id}/taskInstances/{task_id}/xcomEntries

HTTP Method

GET

Response Type

XComCollection

Parameters

  • Path: dag_id String

    The DAG ID.

  • Path: dag_run_id String

    The DAG Run ID.

  • Path: task_id String

    The Task ID.

  • Query: limit Integer

    The numbers of items to return.

  • Query: offset Integer

    The number of items to skip before starting to collect the result set.
