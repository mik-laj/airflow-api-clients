updatePool

URL

http://localhost/api/v1/pools/{pool_name}

HTTP Method

PATCH

Response Type

Pool

Parameters

  • Path: pool_name String

    The Pool name.

  • Body: Pool Pool(Pool)

  • Query: update_mask List(String)

    The fields to update on the connection (connection, pool etc). If absent or empty, all modifiable fields are updated. A comma-separated list of fully qualified names of fields.
